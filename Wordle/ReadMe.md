# Wordle-spil

Gæt det 5-bogstavers ord

Du har 6 forsøg på at gætte ordet.  
Hvert forsøg vil fortælle dig, om bogstaverne i dit gæt findes i det rigtige ord.

- 0 -> Ikke til stede  
- 1 -> Til stede, men på forkert position  
- 2 -> Til stede på korrekt position  

## Eksempel

Rigtigt ord: REVEL  
Gættet ord: OTTER

- e ER KORREKT  
- r findes et sted i ordet  

Positionskode: 0, 0, 0, 2, 1

==English==

# Wordle game

Guess the 5 letter word

You have 6 tries to guess the word.
Each attempt will tell you if the letters from your chosen word are found in the correct word.

- 0 -> Not present
- 1 -> Present but on the wrong position
- 2 -> Present on the correct position

## Example:
Correct word: REVEL
Guessed word: OTTER

- e IS CORRECT
- r Is foundable in the word

Showed position: 0, 0, 0, 2, 1
